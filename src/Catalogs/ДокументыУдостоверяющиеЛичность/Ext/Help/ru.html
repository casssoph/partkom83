Документы, удостоверяющие личность

Для каждого физического лица можно указать реквизиты документа, удостоверяющего его личность. Для указания вида документа и используется этот справочник.

Справочник может быть заполнен при помощи классификатора налоговых органов и ПФР; для этого в форме списка справочника предназначена кнопка «Подбор».
