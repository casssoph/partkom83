Документ «Заявка покупателя» и Документ «Корректировка заявки покупателя»

Документ имеет 2 вида операций:

- "Заявка покупателя" - заявка на поставку товара под покупателя, оформленная на сайте или интерактивно

- "Пополнение склада" - пополнение складских остатков

 

Особенности заполнения документов

Шапка документа заполняется автоматически значениями по умолчанию, установленными для пользователя (см. регистр сведений "Настройки пользователей") .

При заполнении табличной части "Товары" обязательным для заполнения полем является "Прайс поставщика", определяюще поставщика и цены, по которым в дальнейшем будет сформирован заказ.

 

Особенности проведения по регистрам

Регистр накопления "Заявки покупателей"

В регистр записи делаются по каждой строке заявки, которые отражают текущее состояние строки заявки. 

 

Регистр накопления "Резервы товаров"

Документ резервирует только те позиции, которые покупатель заказывает с нашего склада. В этом случае в поле "Прайс поставщика" табличной части "Товары" указывается прайс, который соответствует нашему складу (владелец - элемент справочника "Склады").

Движения по регистру будут сфомированы только на то количество, которое фактически возможно зарезервировать по текущий заказ на момент проведения документа. Незарезервированное количество пойдет в отказ, что будет отражено в соответствующих записях регистра "Заявки покупателей".
