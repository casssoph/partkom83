Описание количественных полей

Группа «Заявки покупателя»

  • Заявлено – Количество из регистра «Заявки покупателей», из приходных движений сделанных документами «Заявка покупателя» и «Корректировка заявки покупателя».
  • Отказ – Количество из регистра «Отказы по заявкам» из движений сделанных документами «Заявка покупателя» и «Корректировка заявки покупателя».
  • Итог – Разница между полями «Заявлено» и «Отказ» (Итог = Заявлено - Отказ).

    • Группа «Заказ поставщику»

  • Заказано – Количество из регистра «Заказы поставщикам», из приходных движений сделанных документом «Заказ поставщику».
  • Отказ – Количество из регистра «Отказы по заявкам» из движений сделанных документами «Заказ поставщику» и «Корректировка заказа поставщику».
  • Итог – Разница между полями «Заказано» и «Отказ» (Итог = Заказано - Отказ).

    • Группа «Поступление товаров услуг»

  • Поступило – Количество из табличных частей «Размещение заявок» проведенных документов «Поступление товаров услуг».
  • Отказ – Количество из табличных частей «Причины отказов» проведенных документов «Поступление товаров услуг».
  • Итог – Разница между полями «Поступило» и «Отказ» (Итог = Поступило - Отказ).

    • Группа «Реализация товаров услуг»

  • Количество План – Поле «Количество план» из табличных частей «Товары» проведенных документов «Реализация товаров услуг».
  • Отказ – Количество из табличных частей «Причины отказов» проведенных документов «Реализация товаров услуг».
  • Отгружено – Поле «Количество» из табличных частей «Товары» проведенных документов «Реализация товаров услуг».
  • Итог – Разница между полями «Количество План» и «Отказ» (Итог = Количество План - Отказ).
