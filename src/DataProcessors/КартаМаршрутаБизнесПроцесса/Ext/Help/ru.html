Обработка «Карта маршрута»

Обработка предназначена для схематического описания текущего состояния бизнес-процесса. Каждая точка бизнес-процесса представляется в виде соответствующего элемента схемы.

Возможны следующие варианты выделения точек «действия» на схеме в зависимости от состояния задач в ней:

  элемент схемы  выделяется красной пунктирной рамкой, если есть невыполненные задачи в точке
  элемент схемы будет заштрихован, если все задачи в точке выполнены
  серым цветом выделяются отключенные в настройке точки
  светлый цвет элемента схемы означает, что в данной точке задач нет

При необходимости возможен быстрый переход к задачам  каждой точки маршрута.
