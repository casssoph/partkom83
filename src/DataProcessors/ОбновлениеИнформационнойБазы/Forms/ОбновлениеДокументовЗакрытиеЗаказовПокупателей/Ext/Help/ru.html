Обновление документов "Закрытие заказов покупателей"

Обработка используется в контексте обновления на ред. 10.3

Обработка выполняет перенос данных о закрытых внутренних заказах из документов "Закрытие заказов покупателей" в документы "Закрытие внутренних заказов".

Необходимо выполнить обновление для тех документов "Закрытие заказов покупателей", которые предполагается перепроводить после обновления.

Порядок действий следующий:

  • Указать период, за который необходимо выполнить обновление документов "Закрытие заказов покупателей"
  • Заполнить список обновляемых документов с помощью кнопки "Заполнить"
  • Нажать кнопку "Выполнить".
